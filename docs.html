EXHIBITION

Data on Parade

Documentation

Overview

Exhibition consists of a client and server part. The server combines storage and processing facilities while the client is concerned mostly with user interaction with the server's data.

All data is wrapped with metadata in Exhibition. The structure of data and logical encapsulations that define Exhibition are as follows:

Data
type
String of data's MIME type
parseableAs
String indicating how the data string should be handled
parsed
Object with methods for interacting with the data string
data
String containing the data
